ngxFlowchart

angular module for flow diagrams

how to use

  • install the module using `npm install ngxflowchart`
  • import `IFlowchart` interface from the module - the data passed to the `flowchart` component should be an `IFlowchart`
  • use the `flowchart` tag in the template, pass `data` attribute

demo

click on add node button to add a node. update node names, port names and drag the nodes around, as required
